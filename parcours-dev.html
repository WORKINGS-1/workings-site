Objectif : Développeur Salesforce

Ce parcours vous prépare à développer sur Salesforce en utilisant Apex, Visualforce, Lightning Web Components et les meilleures pratiques de développement.

Idéal pour les profils techniques souhaitant s’orienter ou se spécialiser dans l’écosystème Salesforce.

Ce que vous apprendrez

• Langage Apex & Triggers

• Composants LWC (Lightning Web Components)

• Intégrations API & sécurité

• Déploiement, test et bonnes pratiques
